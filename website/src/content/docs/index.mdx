## Quick MCP Setup

Add imugi to your Claude Code or Cursor config:

```json
{
  "mcpServers": {
    "imugi": {
      "command": "npx",
      "args": ["-y", "imugi-ai", "mcp"]
    }
  }
}
```

Then just ask your AI to implement a design — imugi handles the rest.
