How it works

The Boulder Loop

imugi doesn't just generate code — it verifies, compares, and iterates until the output matches your design pixel-by-pixel.

01
design/
Input your design image — screenshot, Figma export, or hand-drawn sketch.
🎨
02
generate/
AI generates HTML & CSS code from the design using your project's tech stack.
⚡
03
capture/
Headless Chromium captures a pixel-perfect screenshot of the rendered output.
📸
04
compare/
SSIM + pixel diff + Claude Vision analyze every difference with a heatmap.
🔬
05
patch/
Auto-fix discrepancies with surgical patches or full regeneration.
🩹
↻ repeat until score ≥ 95%

MCP Tools

5 tools, one integration

Add imugi as an MCP server and your AI gets superpowers.

Tool
Description
imugi_capture
Screenshot any URL via headless Chromium with configurable viewport
imugi_compare
Compare design vs screenshot — returns SSIM score, pixel diff, and heatmap
imugi_analyze
Classify diff regions and generate actionable CSS fix suggestions
imugi_detect
Auto-detect project framework, CSS method, and language
imugi_serve
Start a dev server for the target project with HMR support

Get Started

Two steps. That's it.

Install globally and add to your AI tool's MCP config.

1
Install
$ npm install -g imugi-ai
2
Add MCP config
{
  "mcpServers": {
    "imugi": {
      "command": "npx",
      "args": ["-y", "imugi-ai", "mcp"]
    }
  }
}
