# imugi Example: Hero Section

Terminal-inspired dark hero section designed in Pencil, implemented in HTML/CSS.

## Design

- Style: Terminal/CLI aesthetic with dark mode
- Colors: `#0A0A0A` background, `#10B981` green accent
- Fonts: JetBrains Mono + IBM Plex Mono
- Features: Nav, hero with CTA, 4-step flow diagram, MCP tools grid, footer

## Run

```bash
npx serve .
```

## Test with imugi

```bash
# Compare design vs implementation
imugi compare ./design.png --screenshot ./screenshot.png

# Or use MCP tools
imugi mcp
```
